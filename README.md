# 🎓 TimeFlow 课程表

&gt; 一款专为大学生设计的精美课程表应用，让学习时光更有序、更优雅。

---

## ✨ 核心特色

### 🎨 莫兰迪色系设计
采用高级莫兰迪配色方案，低饱和度色彩让视觉体验更加舒适，长时间使用也不疲劳。每一节课都有独特的色彩标识，既美观又实用。

### 🎯 精美原创 Logo
专业设计的品牌标识，简约而不简单。Logo 融合时钟与书本元素，象征时间管理与知识积累的完美结合。

### 🚫 永不添加广告
**承诺永久免费，永不添加任何形式的广告。**

没有开屏广告，没有弹窗干扰，没有横幅占用空间。只有纯粹、流畅的课程表体验，让你专注于学习本身。

&gt; 💡 我们的理念：工具应该服务于用户，而不是打扰用户。

### 🔓 开源人人可编辑
**完全开源，代码透明，自由定制。**

- 📖 源码公开，任何人都可以查看、学习
- 🛠️ 支持二次开发，打造专属功能
- 🤝 欢迎提交 PR，共建更好的课程表
- 📜 MIT 协议，商用友好
