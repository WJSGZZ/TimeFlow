# <img src="logo.svg" width="25"/> TimeFlow 光流

一款专为南通大学教务系统设计的精美课程表应用，记录你校园时光的流动轨迹。

## ✨特色
- 无需注册，导入即用
- 数据本地存储
- 无广告
- 桌面小组件
- 上课前30分钟通知
- UI精美，莫兰迪色系
- 冬夏作息一键切换
